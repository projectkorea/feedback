# 📝 PRD (Product Requirements Document)

## 1. 문제 정의

### 1.1 현재 상황

- 개발자들은 사용자 피드백을 수집하기 위해 복잡한 써드파티 서비스를 통합하거나 자체 솔루션을 구축해야 함
- 기존 솔루션들은 무겁고, 비용이 높으며, 커스터마이징이 제한적임

### 1.2 해결하고자 하는 문제

- 앱에 빠르고 쉽게 임베드 가능한 경량 피드백 수집 시스템의 부재
- 프론트엔드 개발자가 최소한의 설정으로 즉시 사용할 수 있는 위젯의 필요성
- 프로덕트 팀이 수집된 피드백을 효과적으로 관리하고 분석할 수 있는 도구의 필요성

---

## 2. 제품 컨셉

### 2.1 핵심 컨셉

피드백 수집을 위한 경량 SDK(바닐라 JS, React)와 Node.js API 서버, MongoDB로 구성된 PoC입니다. 클라이언트는 위젯(UI)을 통해 사용자 피드백을 전송하고, 서버는 검증/레이트 리밋/인증을 거쳐 안전하게 저장합니다. 대시보드(추후)에서 분석·조회·설정을 제공합니다.

- 목적: 앱에 빠르게 임베드 가능한 피드백 수집/설정 위젯 제공
- UX 철학: 빠른 임베드, 가벼운 의존성, 기본 제공 UI로 즉시 사용 가능
- 차별점: 모노레포로 SDK·API 일관 개발, 프로젝트/설정/커스텀 필드까지 한 번에

### 2.2 핵심 기능

#### 필수 기능

1. **SDK 제공**: 바닐라 JS SDK와 React SDK, 플로팅/설정 위젯 및 기본 Modal/Form UI
2. **API 서버**: 피드백 전송/조회, 프로젝트 키 기반 인증, 레이트 리밋, 검증, MongoDB 저장
3. **설정 관리**: 프로젝트별 위젯 설정(위치/테마/브랜딩/이메일 수집)과 커스텀 필드 지원

#### 추가 기능 (Good to have)

- 피드백 대시보드: 수집된 피드백 분류, 검색, 통계 시각화
- 알림 시스템: 새로운 피드백 발생 시 이메일/Slack 알림
- 다국어 지원: 위젯 UI의 다국어 지원

---

## 3. 유저 플로우

### 3.1 개발자 (SDK 통합)

1. 프로젝트 생성 및 API 키 발급
2. SDK 설치 및 초기화 (publicKey, projectId 설정)
3. 앱에 위젯 컴포넌트 추가
4. 배포 후 피드백 자동 수집 시작

### 3.2 최종 사용자 (피드백 제출)

1. 앱에서 플로팅 버튼 또는 설정 메뉴 진입
2. 피드백 폼 작성 (평점, 메시지, 이메일 등)
3. 제출 후 확인 메시지 표시
4. 데이터는 서버에 자동 저장

### 3.3 프로덕트 매니저 (피드백 관리)

- 대시보드에서 수집된 피드백 조회, 필터링, 분석
- 프로젝트별 설정 원격 업데이트 (위젯 스타일, 활성화/비활성화 등)

---

## 4. 타겟 유저

### 4.1 프론트엔드 개발자

- 앱에 위젯을 빠르게 임베드하고 최소 설정으로 피드백 수집을 시작하려는 사용자

### 4.2 프로덕트 매니저

- 수집된 피드백을 분류·검색·통계로 확인하고 설정을 원격으로 제어하려는 사용자

### 4.3 스타트업/프로덕트 팀

- PoC 단계에서 가벼운 설치로 실제 사용자 의견을 조기에 모으려는 팀

---

## 5. 성공 지표 (KPI)

### 5.1 핵심 지표

- **통합 시간**: 개발자가 SDK를 앱에 통합하는 데 걸리는 시간 (목표: 10분 이내)
- **피드백 수집률**: 앱 사용자 대비 실제 피드백 제출 비율

### 5.2 제품 지표

- **SDK 다운로드 수**: npm 패키지 다운로드 횟수
- **활성 프로젝트 수**: SDK를 활발히 사용 중인 프로젝트 수
- **평균 응답 시간**: API 서버의 평균 응답 속도

---

## 6. 향후 계획

### 6.1 단기 목표

- PoC 완성: SDK(바닐라/React), API 서버, 기본 위젯 UI 개발
- 로컬 개발 환경 구축: Docker Compose로 MongoDB + API 서버 통합

### 6.2 중기 목표

- 대시보드 개발: 피드백 조회/분류/통계 기능
- 알림 시스템: 이메일/Slack 통합
- 다국어 지원

### 6.3 장기 목표

- 엔터프라이즈 기능: 팀 협업, 권한 관리, 고급 분석
- 플러그인 시스템: 커스텀 위젯/폼 확장
