# docs - follow the format

- 작업 완료 후 반드시 확인
- 기존 문서 구조가 새로운 섹션 유지되없이 업데이트 되었는가?

## 문서 구조 절대 고정

- .claude/ 폴더의 파일 생성/삭제/이름변경 절대 금지
- 각 문서의 헤딩 구조 변경 금지 (## 레벨)
- 기존 섹션 내용만 업데이트
- 새로운 헤딩 추가, 기존 헤딩 수정 금지
- '현재 상태', '기술적 현황', '참고사항' 등 임의 헤딩 섹션 생성 금지
- 이모지, 체크박스 등 장식 요소 추가 금지

## .claude/todo.md

```
# Todo
## 현재 목표
## 추가 기능 리스트
```

## .claude/CLAUDE.md

```
# Context
## 핵심 컨셉
## 타겟 유저
## 기술 스택
### 개발 환경
### 핵심 라이브러리
### 추가 라이브러리
## 상태 관리 구조
## 아키텍처 설계
```

룰에 따라 문서를 업데이트하세요.
