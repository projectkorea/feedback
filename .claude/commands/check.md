# check - 커밋 전 코드 검토

coderules.md 체크포인트에 따라 최근 작업을 검토하고 수정해주세요.

## 체크리스트

- [ ] **기존 패턴 따름?** - 프로젝트 구조와 일관성 확인
- [ ] **불필요한 코드 없음?** - 사용하지 않는 import, 함수, 변수 제거
- [ ] **최소한으로 구현?** - 과도한 추상화나 복잡성 제거
- [ ] **추상화 필요성 명확?** - 추상화한 부분이 실제로 필요한지 확인
- [ ] **변수/함수명 직관적?** - 역할이 이름에서 바로 드러나는지 확인
- [ ] **책임 분리 명확?** - 각 함수/클래스가 단일 책임을 갖는지 확인

## 실행 순서

1. 최근 변경된 파일들 확인
2. 각 파일별로 체크리스트 적용
3. 문제 발견시 즉시 수정
4. 수정 후 다시 체크
